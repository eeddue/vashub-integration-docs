# Player Info

This request is used to request for player details.

| path    | **`/info`**     |
| ------- | --------------- |
| method  | **`POST`**      |
| headers | **`x-api-key`** |

### Request

<Tabs>
  <TabItem value="parameters" label="Parameters">
    | Parameter   | Description |
    | ------ | ------- |
    | user_id | **String** - Player's id on the operator's side  |
    | session_token | **String** - Token generated by provider for current game session |
    | currency | **String** - Game currency. (currency code) |
  </TabItem>
  
  <TabItem value="example" label="Example">

    ```json
    {
      "user_id": "180",
      "session_token": "01d9c4ce7180589176d7f3f4f7b7581243bd450dad44f33029196c3bfc5e9771",
      "currency": "USD"
    }
    ```

  </TabItem>

</Tabs>

### Response

<Tabs>
  <TabItem value="parameters" label="Parameters">

    | Parameter | Description                                                                                         |
    | --------- | --------------------------------------------------------------------------------------------------- |
    | code      | **Number** - response status code                                                                       |
    | message   | **String** - response explanation                                                                       |
    | data      | **Object** <ul><li>user_id - ***String*** player id</li><li>username - ***String*** player name</li><li>balance - ***Number*** player balance</li><li>currency - ***String*** currency code</li></ul> |

  </TabItem>

  <TabItem value="success" label="Success Example">

    ```json
    {
      "code": 200,
      "message": "ok",
      "data": {
        "user_id": "180",
        "username": "Dan321",
        "balance": 49992,
        "currency": "GEL"
      }
    }
    ```

  </TabItem>

  <TabItem value="error" label="Error Example">

    ```json
    {
        "code": 401,
        "message": "Invalid user",
        "data": {}
    }
    ```

  </TabItem>
</Tabs>
